# Excel-Road-Accident-Dashboard
📌 Project Workflow
1. Requirement Gathering
Defined business requirements and project objectives.
Identified primary and secondary KPIs.
Selected the most effective charts and visualizations for the dashboard.
2. Data Cleaning
Cleaned and prepared the raw dataset containing approximately 307,000 records.
Removed inconsistencies, corrected data formats, and standardized column values.
Ensured data quality for accurate analysis.
3. Data Processing
Created calculated columns using Excel formulas.
Extracted features such as Year from the Date column.
Prepared the dataset for advanced reporting and analysis.
4. Data Analysis
Built multiple Pivot Tables to summarize and analyze the data.
Used calculated fields/items to group related categories (e.g., vehicle types and road surface conditions).
Generated meaningful insights from the processed dataset.
5. Data Visualization

Created interactive visualizations, including:

🍩 Donut Charts for accident severity distribution.
📈 Line Charts for monthly accident trends.
🌳 Tree Maps for road surface condition analysis.
📊 Additional charts to highlight key performance indicators.
6. Dashboard Development
Designed a professional and user-friendly dashboard layout.
Created KPI cards with custom formatting.
Organized charts and visuals for better readability.
Applied consistent colors and formatting for an attractive interface.
7. Interactive Filters
Added Slicers and Timeline controls.
Connected filters to all Pivot Tables using Report Connections.
Enabled dynamic and interactive dashboard exploration.
8. Report Navigation
Created a dedicated Data Analysis worksheet for source Pivot Tables.
Added navigation buttons and hyperlinks for seamless user experience.
Structured the workbook for easy maintenance and scalability.

Microsoft Excel
Pivot Tables
Pivot Charts

Excel Formulas
Conditional Formatting
Dashboard Design Techniques
