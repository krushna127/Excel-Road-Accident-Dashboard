<h2>📌 Project Workflow</h2>

<h3>1. Requirement Gathering</h3>
<ul>
    <li>Defined business requirements and project objectives.</li>
    <li>Identified primary and secondary KPIs.</li>
    <li>Selected the most effective charts and visualizations for the dashboard.</li>
</ul>

<h3>2. Data Cleaning</h3>
<ul>
    <li>Cleaned and prepared the raw dataset containing approximately <strong>307,000 records</strong>.</li>
    <li>Removed inconsistencies, corrected data formats, and standardized column values.</li>
    <li>Ensured data quality for accurate analysis.</li>
</ul>

<h3>3. Data Processing</h3>
<ul>
    <li>Created calculated columns using Excel formulas.</li>
    <li>Extracted features such as <strong>Year</strong> from the Date column.</li>
    <li>Prepared the dataset for advanced reporting and analysis.</li>
</ul>

<h3>4. Data Analysis</h3>
<ul>
    <li>Built multiple <strong>Pivot Tables</strong> to summarize and analyze the data.</li>
    <li>Used calculated fields and grouped related categories (e.g., vehicle types and road surface conditions).</li>
    <li>Generated meaningful insights from the processed dataset.</li>
</ul>

<h3>5. Data Visualization</h3>
<ul>
    <li>🍩 Donut Charts for accident severity distribution.</li>
    <li>📈 Line Charts for monthly accident trends.</li>
    <li>🌳 Tree Maps for road surface condition analysis.</li>
    <li>📊 Additional charts to highlight key performance indicators.</li>
</ul>

<h3>6. Dashboard Development</h3>
<ul>
    <li>Designed a professional and user-friendly dashboard layout.</li>
    <li>Created KPI cards with custom formatting.</li>
    <li>Organized charts and visuals for better readability.</li>
    <li>Applied consistent colors and formatting for an attractive interface.</li>
</ul>

<h3>7. Interactive Filters</h3>
<ul>
    <li>Added <strong>Slicers</strong> and <strong>Timeline</strong> controls.</li>
    <li>Connected filters to all Pivot Tables using <strong>Report Connections</strong>.</li>
    <li>Enabled dynamic and interactive dashboard exploration.</li>
</ul>

<h3>8. Report Navigation</h3>
<ul>
    <li>Created a dedicated <strong>Data Analysis</strong> worksheet for source Pivot Tables.</li>
    <li>Added navigation buttons and hyperlinks for seamless user experience.</li>
    <li>Structured the workbook for easy maintenance and scalability.</li>
</ul>

<h2>🛠️ Tools Used</h2>
<ul>
    <li>Microsoft Excel</li>
    <li>Pivot Tables</li>
    <li>Pivot Charts</li>
    <li>Slicers &amp; Timeline</li>
    <li>Excel Formulas</li>
    <li>Conditional Formatting</li>

</ul>
